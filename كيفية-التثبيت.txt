تقرير وتسليم وردية الكاشير — تطبيق ويب قابل للتثبيت (PWA)
===========================================================

الاستخدام السريع (بدون تثبيت):
  افتح ملف index.html بأي متصفّح. يعمل بالكامل بدون إنترنت.

التثبيت كتطبيق على الجهاز (مع أيقونة وشاشة كاملة):
  التثبيت يتطلب فتح التطبيق عبر رابط (http/https) وليس كملف مباشر.
  أسهل الطرق:

  1) استضافة مجانية (يظهر زر تثبيت على أندرويد والكمبيوتر):
     - ارفع محتويات هذا المجلد كاملاً إلى أي استضافة ملفات ثابتة، مثل:
       Netlify Drop ( app.netlify.com/drop ) — اسحب المجلد وأفلته، ينتج رابط فوراً.
       أو GitHub Pages / Cloudflare Pages / Vercel.
     - افتح الرابط بمتصفّح Chrome/Edge ← قائمة المتصفّح ← «تثبيت التطبيق».

  2) خادم محلي على نفس الجهاز/الشبكة (للأجهزة الداخلية):
     - داخل هذا المجلد شغّل أمراً بسيطاً مثل:  python -m http.server 8080
     - افتح  http://localhost:8080  ثم ثبّته من قائمة المتصفّح.

  3) آيفون/آيباد (Safari):
     - افتح الرابط في Safari ← زر المشاركة ← «إضافة إلى الشاشة الرئيسية».
       ستظهر الأيقونة ويفتح التطبيق بملء الشاشة.

النسخ الاحتياطي:
  البيانات تُحفظ داخل المتصفّح. للاحتفاظ بنسخة دائمة على الجهاز استخدم
  زر «نسخة احتياطية» لتنزيل ملف .json، و«استيراد» لإرجاعه لاحقاً.
  يمكن أيضاً تصدير السجل Excel (CSV) أو PDF.

ملاحظة: احتفظ بكل ملفات المجلد مع بعضها (index.html و sw.js و manifest والأيقونات).
