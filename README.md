# D.Report — تقرير وتسليم وردية الكاشير

تطبيق ويب (PWA) لإعداد وطباعة تقرير تسليم وردية الكاشير، يعمل بالكامل داخل المتصفّح
بدون خادم، ويحفظ البيانات محلياً على الجهاز.

## المحتويات
- index.html ............ التطبيق كاملاً (HTML/CSS/JS في ملف واحد)
- manifest.webmanifest .. إعدادات تثبيت التطبيق
- sw.js ................. Service Worker (تحديث تلقائي network-first)
- icon-192.png / icon-512.png / icon-maskable.png / apple-touch-icon.png .. أيقونات التطبيق
- icon-source.svg ....... مصدر الشعار (SVG) للتعديل لاحقاً
- كيفية-التثبيت.txt ...... تعليمات النشر والتثبيت
- cashier-report-standalone.html .. نسخة مفردة تعمل بمجرد فتحها (بلا استضافة)

## المزايا
- حساب تلقائي: إجمالي المبيعات − طرق الدفع (مدى/آجل/تابي/تمارا/تحويلات…) = نقدي،
  ثم المصروفات والإيرادات والنقد المتوقع مقابل المعدود، مع كشف العجز/الزيادة.
- تقويم عربي (تنقّل شهور/سنوات) ومنتقي وقت احترافي.
- قوائم ذكية لأسماء الكاشير والمستلمين وطرق الدفع (تُحفظ وتُضاف بسهولة).
- سجل ورديات + عرض/تعديل + طباعة 80مم (Epson TM-T20III).
- تصدير Excel / PDF + نسخة احتياطية JSON (يدوية) ونسخ تلقائي لمجلد (Chrome/Edge).
- تذكير إقفال الجلسة بعد الحفظ، وتثبيت وقت الحفظ/التعديل الفعلي على الإيصال.

## النشر (GitHub Pages)
ارفع محتويات هذا المجلد إلى مستودع Public، ثم فعّل Settings ← Pages ←
Deploy from a branch ← main ← /(root). للتحديث لاحقاً: ارفع index.html فوق القديم.
(غيّر رقم الإصدار في sw.js فقط إذا أضفت/غيّرت ملفات).
